
# What is lexicon?

A repository that contains some lexicon of computer science, mathematics and anything
interesting. Lexicon may contain a small phrase.


## Characteristics of Lexicons

1. Lexicons should not be very common.
2. Lexicons should be relevant.
3. The concept that the lexicon implies is interesting.
